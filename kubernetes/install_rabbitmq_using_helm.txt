Cek Helm (apakah sudah terinstall atau belum) :
helm

Cek List Helm :
helm list

Search Repo RabbitMQ :
helm search repo rabbitmq

Tambahkan repo bitnami :
helm repo add bitnami https://charts.bitnami.com/bitnami

Update Repo :
helm repo update
